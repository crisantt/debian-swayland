# Debian Wayland Using Sway

Install git first

```
sudo apt install git
```

### Install Sway Using Debian Package Manager
```
mkdir -p ~/github && cd ~/github
git clone https://github.com/crisantt/debian-swayland.git
cd debian-swayland
bash install.sh
```
